## 4. 함수와 함수형 프로그래밍
-------------

* [HOME](./README.md)

* **함수의 선언**

  ```kotlin
  fun sum(a: Int, b: Int):Int
   {
     return a+b
   }
  
  ```
  
  * 코틀린에서 함수는 `fun`키워드로 선언합니다.
  * 함수의 이름을 작성하고 그 옆에 **함수의 매개변수**를 선언해줍니다.
    * 함수의 매개변수는 **<변수의 이름: 자료형>** 형식으로 선언되며 생략 가능합니다.
  * 매개변수 선언 후 반환타입을 명시해주며 반환타입이 `Unit`일 경우 생략 가능합니다.
